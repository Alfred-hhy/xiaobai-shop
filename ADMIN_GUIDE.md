# 小白衣橱 - 管理后台操作手册

## 系统概述

小白衣橱是一个基于 uni-app (Vue3) 的小个子女装 H5 展示商城。
后端使用 Supabase（数据库 + 存储 + 认证），部署在 GitHub Pages。

**线上地址**: https://alfred-hhy.github.io/xiaobai-shop/

---

## 一、进入管理后台

## 二、商品管理

### 2.1 商品列表

登录后看到所有商品（包括已下架的），每个商品显示：
- 商品图片（第一张）
- 商品名称
- 状态标签（绿色「在架」/ 橙色「下架」）
- 售价
- 操作按钮

### 2.2 新增商品

1. 点击右上角 **「+ 新增」**
2. 填写商品信息：

| 字段 | 必填 | 说明 |
|------|------|------|
| 商品名称 | 是 | 商品标题 |
| 分类 | 是 | 下拉选择：连衣裙/上衣/裤装/半裙/健康好物 |
| 售价 | 是 | 当前售价（数字） |
| 原价 | 否 | 划线价，留空则不显示 |
| 商品描述 | 否 | 最多500字 |
| 可选尺码 | 否 | 英文逗号分隔，如 `XS,S,M,L` |
| 可选颜色 | 否 | 英文逗号分隔，如 `黑色,白色` |
| 新品标签 | 否 | 点击切换开/关 |
| 热销标签 | 否 | 点击切换开/关 |
| 商品图片 | 否 | 最多5张，从相册选择或拍照上传 |
| 上架状态 | 否 | 默认开启，关闭后前台不显示 |

3. 点击 **「创建商品」** 保存

### 2.3 编辑商品

1. 在商品列表点击 **「编辑」**
2. 修改需要更改的字段
3. 点击 **「保存修改」**

### 2.4 上架 / 下架

- 在商品列表直接点击 **「下架」** 或 **「上架」** 按钮
- 下架后：商品在前台（首页、分类页）不可见，但管理后台仍可看到
- 上架后：商品恢复在前台显示

### 2.5 删除商品

1. 点击 **「删除」** 按钮
2. 弹出确认框，点击「确定」
3. **注意：删除不可撤销**，商品数据永久删除

### 2.6 退出登录

点击右上角 **「退出」** 按钮。

## 三、分类管理

### 3.1 进入分类管理

在商品管理页面，点击右上角 **「分类」** 按钮。

### 3.2 新增分类

1. 点击 **「+ 新增」**
2. 填写：
   - **分类ID**（必填）：英文标识，如 `jacket`，创建后不可修改
   - **分类名称**（必填）：中文名，如 `外套`
   - **图标**：emoji，如 `🧥`
   - **排序**：数字越小越靠前
3. 点击 **「保存」**

### 3.3 编辑分类

点击 **「编辑」**，可修改名称、图标、排序（ID 不可改）。

### 3.4 删除分类

- 点击 **「删除」** → 确认
- **注意**：如果该分类下有商品，删除会失败（外键约束）
- 需先将该分类下的商品改到其他分类

## 七、技术信息（开发者参考）

### 数据库结构

**categories 表**
```
id (TEXT, PK) | name (TEXT) | icon (TEXT) | sort_order (INT)
```

**products 表**
```
id (SERIAL, PK) | name | price | original_price | category (FK→categories.id)
description | sizes (JSONB) | colors (JSONB) | images (JSONB)
is_new (BOOL) | is_hot (BOOL) | status ('on'/'off')
created_at | updated_at (自动更新)
```
